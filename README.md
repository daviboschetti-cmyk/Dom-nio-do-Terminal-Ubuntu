## 📃 Documentação : Domínio do Terminal Ubuntu


## 📕 Lista de Comandos mais usados

--> pwd: Exibi o caminho da pasta atual

--> ls: Lista de arquivos e Pastas

--> cd: Entra em uma pasta

--> mkdir: Cria pasta

--> touch: Cria arquivos ou pastas 

--> nano/vim: Editores de textos internos


## 💡DIcas práticas

--> Use Tab sempre que possível para autocompletar pastas e arquivos

--> Antes de criar algo, use ls para evitar arquivos duplicados.

-->Use pwd com frequência para não se perder nas pastas.

-->Crie tudo organizado desde o início.

-->Ao usar cp e mv, confira com ls se funcionou corretamente.

-->No Nano, observe os atalhos na parte de baixo da tela.

-->No Vim use o "i" para escrever e "Esc" para sair do modo de edição.

-->Sempre salve antes de sair dos editores.

-->Use nomes simples e sem espaços nos arquivos (style.css).


## ❌ Dificuldades Encontradas ao usar o Vim ou Nano 

Para um estudante, usar editores de terminal como o Vim e o Nano pode ser desafiador no início.

 O Vim costuma causar confusão por funcionar com diferentes modos e comandos pouco intuitivos, o que gera perda de tempo em tarefas simples.
 
  Já o Nano é mais fácil de usar, pois mostra os comandos na tela, sendo mais facíl para iniciantes. 
  
  Com o tempo e a prática, porém, o estudante pode perceber que o Vim é mais eficiente e poderoso, enquanto o Nano continua sendo uma opção prática para edições rápidas.
